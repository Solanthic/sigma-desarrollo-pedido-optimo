## Key Schema Discovery: SOP_PAC_KPI_CITY_CLUB

This table was queried live and revealed a **retailer-specific version of the core pipeline variables**:

```
SAP | SKU | VENTA_PROMEDIO | DIA_ENTREGA | TRANSITO_1 | TRANSITO_2 | 
PERIODO_TRANSITO1 | PERIODO_TRANSITO2 | VN_PK | POR_MERMA | TIPO_MERMA | 
VOL_PROMEDIO | PRODUCTO | FAMILIA
```

**This confirms**: The same core variables (TIPO_MERMA, VENTA_PROMEDIO, VOL_PROMEDIO, transit data) are computed for City Club and Sam's Club stores, not just the main pipeline. These retailer-specific KPI tables are **pre-computed pipeline outputs** for specific chains, containing the same business logic as T05+T07.
